# Realisieren Pulse UI Redesign Prompt

Redesign the Realisieren Pulse web app as a polished professional workforce/productivity dashboard using Lexend Deca as the primary font. The interface should feel like a serious SaaS operations tool: clean, structured, calm, and premium, not decorative or plain.

Use a consistent light dashboard system with a restrained palette: soft app background, white surfaces, charcoal text, indigo primary actions, green success, amber warning, and red danger. Replace casual emoji-style UI controls with compact professional markers, improve sidebar hierarchy, tighten card spacing, reduce overly rounded corners, and add subtle borders/shadows only where they help scanning. Login should feel branded and modern, while internal pages should prioritize dense readable information, clear cards, useful status badges, and professional table/form styling.

Keep the first pass scoped to frontend styling and presentation polish: global typography, shared visual tokens, sidebar/cards/buttons/forms, login hero/form polish, and obvious UI cleanup. Do not change backend behavior or API contracts.
